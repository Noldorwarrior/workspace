# Матрица пресетов верификации

## Пресеты малой верификации (М1–М5)

Малая верификация включается в любую полную автоматически.

| Пресет | Назначение | Механизмы | Скрипты |
|--------|------------|-----------|---------|
| М1 «Базовая» | Минимальная проверка любого документа | №1, №2 | — (LLM only) |
| М2 «Расширенная» | Документы с цифрами и форматированием | №1, №2, №3, №4, №5 | verify_sums, verify_docx_format |
| М3 «Максимальная малая» | Сложные документы с хронологией | №1, №2, №3, №4, №5, №6, №7 | verify_sums, verify_docx_format, verify_dates |
| М4 «Презентационная» | Презентации pptx/html | №1, №2, №8, №9 | verify_pptx_format, verify_pptx_html_sync |
| М5 «Стратегическая малая» | Стратегические документы | №1, №2, №10, №11 | — (LLM only) |

## Правила комбинирования пресетов

1. **Малая + полная** — всегда. Малая включается автоматически.
2. **Два полных пресета** — можно. Механизмы объединяются, дубли удаляются.
3. **Три и более полных** — можно, но рекомендуется режим `parallel`.
4. **П5 «Максимум»** — включает всё. Комбинировать с другими бессмысленно.
5. **Выбор малой**: определяется автоматически по типу выходного файла:
   - docx → М2 «Расширенная»
   - xlsx → М2 «Расширенная»
   - pptx/html → М4 «Презентационная»
   - стратегия/концепция → М5 «Стратегическая малая»
   - прочее → М1 «Базовая»
